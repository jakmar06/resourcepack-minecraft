# Zalecane pluginy

## Podstawa Serwera

1. EssentialsX - [https://essentialsx.net/downloads.html](https://essentialsx.net/downloads.html)

{% hint style="info" %}
Core pod twój serwer, posiada większość podstawowych funkcji, przykładowo: obsługa ekonomii vault, czy vanish.
{% endhint %}

{% hint style="warning" %}
Wymagania:&#x20;

* Minimalna wersja serwera: 1.19.4
* Java 17
{% endhint %}

2. Akropolis - [https://modrinth.com/plugin/akropolis](https://modrinth.com/plugin/akropolis/versions)

{% hint style="info" %}
Core pod twój serwer lobby, zawiera automatyczne wiadomości, tryb pvp, wybór serwera, czy blokada wybranych interakcji, np. komend czy niszczenia bloków.
{% endhint %}

{% hint style="warning" %}
Wymagania:&#x20;

* Minimalna wersja serwera: 1.21.10
* Java 21
{% endhint %}

3. Huskhomes - [https://modrinth.com/plugin/huskhomes](https://modrinth.com/plugin/huskhomes)&#x20;

{% hint style="info" %}
Plugin rozszerzający system teleportacji, dodający tworzenie domów przez graczy, warpów, losowy teleport (RTP) oraz  /spawn.
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera: 1.17.1
* Java 21
{% endhint %}

4. WorldEdit - [https://modrinth.com/plugin/worldedit](https://modrinth.com/plugin/worldedit)

{% hint style="info" %}
Narzędzie administracyjne do zarządzania terenem np. do kopiowania terenu, wklejania schematów czy podmieniania bloków.
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera: 1.21.4
* Java 21
{% endhint %}

5. FastAsycWorldEdit - [https://modrinth.com/plugin/fastasyncworldedit](https://modrinth.com/plugin/fastasyncworldedit)

{% hint style="info" %}
Fork WorldEdita do szybkich edycji dużych obszarów, oferujący dodatkowe funkcje dla budowniczych. \
Nie jest zalecany do stosowania na serwerach produkcyjnych, może spowodować nieodwracalne problemy z światem. (Szansa na to jest bardzo niska, choć ryzyko dalej jest na to większe niż przy oryginalnym worldedicie)
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera: 1.21.4
* Java 21
{% endhint %}

6. WorldGuard - [https://modrinth.com/plugin/worldguard](https://modrinth.com/plugin/worldguard/)

{% hint style="info" %}
Narzędzie administracyjne do zarządzania regionami, pozwalający przykładowo na zabezpieczenie spawnu przed niszczeniem.
{% endhint %}

{% hint style="warning" %}
Wymagania:&#x20;

* Minimalna wersja serwera 1.21.11
* Java 21
* [https://modrinth.com/plugin/worldedit](https://modrinth.com/plugin/worldedit) lub [https://modrinth.com/plugin/fastasyncworldedit](https://modrinth.com/plugin/fastasyncworldedit)
{% endhint %}

7. ViaVersion - [https://modrinth.com/plugin/viaversion](https://modrinth.com/plugin/viaversion)

{% hint style="info" %}
Plugin pozwalający na dołączanie graczy z wyższych wersji niż bazowa wersja serwera.
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera 1.7
* Java 17
{% endhint %}

8. ViaBackwards - [https://modrinth.com/plugin/viabackwards](https://modrinth.com/plugin/viabackwards)

{% hint style="info" %}
Plugin pozwajający na dołączanie graczy z niższych wersji niż bazowa wersja serwera. (Minimalna wersja clienta: 1.9.x)
{% endhint %}

{% hint style="warning" %}
Wymagania:&#x20;

* Minimalna wersja serwera 1.8
* Java 17
* [https://modrinth.com/plugin/viaversion](https://modrinth.com/plugin/viaversion)
{% endhint %}

9. ViaRewind - [https://modrinth.com/plugin/viarewind](https://modrinth.com/plugin/viarewind)

{% hint style="info" %}
Plugin pozwalający na dołączanie graczy z niższych wersji niż bazowa wersja serwera.&#x20;

(Minimalna wersja clienta: 1.7.x)
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera 1.9
* Java 17
* [https://modrinth.com/plugin/viabackwards](https://modrinth.com/plugin/viabackwards)
{% endhint %}

10. Carbon Chat - [https://modrinth.com/plugin/carbon](https://modrinth.com/plugin/carbon)

{% content-ref url="carbon-chat.md" %}
[carbon-chat.md](carbon-chat.md)
{% endcontent-ref %}

{% hint style="info" %}
Plugin do zarządzania chatem, posiada wsparcie wielu instancji (synchronizacja chatu między wieloma instancjami), formatowanie chatu poprzez [minimessage](https://docs.papermc.io/adventure/minimessage/format/), oraz kanały chatu.
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera 1.21.4
* Java 21
{% endhint %}

11. CrazyCrates - [https://modrinth.com/plugin/crazycrates](https://modrinth.com/plugin/crazycrates)

{% hint style="info" %}
Aktywnie rozwijany plugin do tworzenia skrzynek z nagrodami.
{% endhint %}

{% hint style="warning" %}
Wymagania:&#x20;

* Minimalna wersja serwera 1.21.8
* Java 21
{% endhint %}

12. ExcellentCrates - [https://modrinth.com/plugin/excellentcrates](https://modrinth.com/plugin/excellentcrates)

{% hint style="info" %}
Aktywnie rozwijany plugin do tworzenia skrzynek z nagrodami z łatwiejszą konfiguracją.
{% endhint %}

{% hint style="warning" %}
Wymagania:

* Minimalna wersja serwera: 1.21.8
* Java 21
* [https://modrinth.com/plugin/nightcore](https://modrinth.com/plugin/nightcore)
* Opcjonalnie [https://modrinth.com/plugin/packetevents](https://modrinth.com/plugin/packetevents)
{% endhint %}
