# Czym jest Minecraft Wiki?

Minecraft Wiki jest ogólną Wikipedią dla właścicieli serwerów Minecraft.

#### Co znajdę na stronie?

* Poradniki dla resourcepacków.
* Informacje o tym czego unikać.
* Poradniki dla pluginów, razem z wyjaśnieniem ich konfiguracji, spisem komend i ich permisji.
* Poradniki dla konfiguracji sieci serwerów.
* Listę popularnych pluginów zebraną w jedno miejsce, oraz linki do ich oficjalnych dokumentacji.
* Listę stron, które pomogą ci w konfiguracji twojego serwera, zarządzania nim bądź administracją.
